# 📸 SnapRoll (스냅롤)
> **찰나의 순간을 스와이프로 즐기는 사진 공유 플랫폼**

스냅롤은 고화질 사진을 감상하며 촬영 정보(EXIF)를 공유하는 안드로이드 애플리케이션입니다. 릴스나 쇼츠처럼 직관적인 스와이프 인터페이스를 통해 사진을 탐색하고, 사진 속 숨겨진 디테일(ISO, 조리개, 셔터스피드 등)을 확인해 보세요.

---

## 주요 기능

### 스냅
앱의 핵심 기능으로, 사진을 한 장씩 넘겨보며 몰입감 있는 감상을 제공합니다.
* 스와이프 탐색: 상/하 제스처를 통해 끊김 없이 새로운 사진을 발견할 수 있습니다.
* 좋아요: 마음에 드는 사진에 반응을 남길 수 있습니다. (로그인 필수)
* 상세 정보 보기: 정보 버튼 클릭 시 해당 사진의 **ISO, 조리개 값, 셔터스피드**를 즉시 확인할 수 있습니다.

### 사진 등록
자신만의 감성이 담긴 사진을 커뮤니티에 공유할 수 있습니다.
* 사진 업로드를 위해서는 반드시 로그인이 필요합니다.
* 업로드할 사진의 메타 데이터 정보(노출의 3요소) 등을 추가할 수 있습니다.

### 마이페이지
사용자 개인 활동을 관리하는 공간입니다.
* 인증 관리: 로그인 및 로그아웃 기능을 제공합니다.
* 활동 확인: 내가 업로드한 사진 리스트와 내 정보를 확인할 수 있습니다.

### 네비게이션
하단 바텀 네비게이션을 통해 주요 화면을 빠르게 전환합니다.
* **좌측**: 사진 등록 | **중앙**: 스냅 (메인 피드) | **우측**: 마이페이지

## 기술 스택

### **Android**
* **Language**: `Kotlin`
* **UI**: `Jetpack Compose`
* **Library**: `Jetpack` (ViewModel, Navigation 등), `Retrofit2` (API 통신), `Coil` (이미지 로딩)

### **Backend & Database**
* **Infrastructure**: `Supabase` (Auth, Realtime Database, Storage)
* **SDK**: `Supabase Kotlin SDK`

### **Architecture**
* **Pattern**: `MVVM` + `Clean Architecture`
* **Dependency Injection**: `Hilt` (추천)
* **Packaging**: `Multi-Module`
